Math Formulas Every Student Should Know

Here are essential math formulas that are useful for school exams and real-life applications. Memorizing them can make solving math problems faster and easier.

1. Area Formulas

Rectangle: A = l × w
Triangle: A = ½ × b × h
Circle: A = π × r²

2. Perimeter Formulas

Rectangle: P = 2(l + w)
Triangle: P = a + b + c
Circle: P = 2πr

3. Algebraic Formulas

(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
a² - b² = (a + b)(a - b)

4. Pythagorean Theorem

For right triangles: a² + b² = c²

5. Speed, Distance, Time

Speed = Distance ÷ Time
Distance = Speed × Time
Time = Distance ÷ Speed

These formulas are building blocks for math success. Keep a printed copy or bookmark this page to use as a quick reference guide!
